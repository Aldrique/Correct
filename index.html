Le Langage HTML

HTML signifie HyperText Markup Language. C'est le squelette de toute page web.

Structure de base

Voici les éléments essentiels :

  • La déclaration <!DOCTYPE html>
  • La balise <html> avec l'attribut lang
  • Le <head> contenant les métadonnées
  • Le <body> pour le contenu visible

Balises de texte

HTML propose diverses balises sémantiques :

  1. <h1> à <h6> pour les titres
  2. <p> pour les paragraphes
  3. <strong> et <em> pour l'importance
  4. <blockquote> pour les citations
Le HTML sémantique améliore l'accessibilité et le référencement naturel.

Les Feuilles de Style CSS

CSS permet de séparer la présentation du contenu.

Sélecteurs CSS

Sélecteurs de base

  • * - Sélecteur universel
  • element - Sélecteur d'élément
  • .class - Sélecteur de classe
  • #id - Sélecteur d'ID

Sélecteurs combinés

  • A B - Descendant
  • A > B - Enfant direct
  • A + B - Voisin direct
  • A ~ B - Voisins généraux

Pseudo-classes

  • :hover - Au survol
  • :focus - En focus
  • :nth-child() - Nième enfant
  • :not() - Négation

Flexbox et Grid

Flexbox

Système de mise en page unidimensionnel.

.container {
    display: flex;
    justify-content: center;
    align-items: center;
}

Grid

Système de mise en page bidimensionnel.

.container {
    display: grid;
    grid-template-columns: 1fr 1fr;
    gap: 20px;
}

Astuce CSS

Utilisez box-sizing: border-box pour un modèle de boîte plus intuitif.

La propriété clamp() permet des tailles de police responsives sans media queries.

Programmation JavaScript

JavaScript est un langage de programmation interprété et orienté objet.

Variables et types

// Déclaration de variables
let nom = "Alice";
const age = 25;
var ancienne = "éviter";

// Types de données
let nombre = 42;           // Number
let texte = "Bonjour";     // String
let actif = true;          // Boolean
let tableau = [1, 2, 3];   // Array
let objet = {nom: "Bob"};  // Object

Fonctions

// Fonction déclarative
function saluer(nom) {
    return `Bonjour, ${nom} !`;
}

// Fonction fléchée
const addition = (a, b) => a + b;

// Fonction anonyme
setTimeout(function() {
    console.log("Exécuté après délai");
}, 1000);

Manipulation du DOM

// Sélection d'éléments
const titre = document.getElementById('titre');
const items = document.querySelectorAll('.item');

// Modification du contenu
titre.textContent = 'Nouveau titre';
titre.innerHTML = 'HTML interprété';

// Gestion des événements
bouton.addEventListener('click', function(e) {
    e.preventDefault();
    console.log('Cliqué !');
});

Exercices pratiques

Exercice 1 : Structure HTML

Créez une page HTML valide avec :

  1. Doctype correct
  2. Balise html avec langue
  3. Head avec meta charset et viewport
  4. Body avec header, main et footer

Exercice 2 : Formulaire complet

Créez un formulaire d'inscription avec :

  • Champs texte, email, password
  • Cases à cocher et boutons radio
  • Liste déroulante
  • Bouton de soumission

Exercice 3 : Tableau de données

Créez un tableau présentant :

  • En-têtes de colonnes
  • Corps avec données
  • Ligne de pied de tableau

Bonnes pratiques d'accessibilité

  • Utiliser des contrastes de couleur suffisants (ratio 4.5:1 minimum)
  • Ajouter des textes alternatifs aux images
  • Structurer avec des titres hiérarchiques (h1 → h6)
  • Permettre la navigation au clavier
  • Utiliser des labels associés aux champs de formulaire
  • Indiquer la langue principale du document
